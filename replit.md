# LEAP Startup Clone

A full-featured marketing website inspired by leapstartup.com, built with React + Vite.

## Tech Stack
- React 18 + TypeScript
- Vite 6 (port 5000)
- React Router v6
- Google Fonts (Inter + Playfair Display)

## Pages
- `/` — Home (hero, stats, features, how it works, courses, testimonials, CTA)
- `/platform` — All programs with detailed module listings and pricing
- `/locations` — 6 Indian city campuses with photos
- `/institutions` — Partnership models for colleges and corporates
- `/community` — Events, alumni spotlight, Discord community
- `/contact` — Application form

## Run
```
npm run dev
```
Server runs on `0.0.0.0:5000`.
